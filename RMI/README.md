# Java RMI

Projekti realizovani koristeći **Java Remote Method Invocation (RMI)** tehnologiju.

## Šta je RMI?

Java RMI omogućava pozivu metoda objekata koji se nalaze na udaljenim JVM instancama, kao da su lokalni. Koristi Java serijalizaciju za prenos podataka.

## Projekti

| Folder | Opis |
|--------|------|
| `src/` | Izvorni kod projekata |

## Pokretanje

1. Kompajlirati sve `.java` fajlove
2. Pokrenuti RMI registry: `rmiregistry &`
3. Pokrenuti server
4. Pokrenuti klijenta
